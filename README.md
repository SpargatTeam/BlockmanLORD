## Compile:

### Download Boost source code and add it on libs and rename the folder on boost

### Download GLFW source and add it on libs

### Download ZLIB source and add it on libs

### Build CMake

```bash
cmake .
```
### Build BlockmanLORD

```bash
make -k
```
